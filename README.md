## <a name="introduction"><img src="https://raw.githubusercontent.com/Tarikul-Islam-Anik/Telegram-Animated-Emojis/main/Smileys/Robot.webp" alt="Robot" width="25" height="25" /> Introduction</a>

**Filipino Mart NC** is a vibrant, responsive web storefront built for a Filipino grocery store based in North Carolina. Built using **React**, **Tailwind CSS**, and **React Router**, the site provides customers with a fast and friendly way to browse and shop for traditional Filipino goods—from classic snacks to pantry essentials.

---

## <a name="tech-stack"><img src="https://raw.githubusercontent.com/Tarikul-Islam-Anik/Animated-Fluent-Emojis/master/Emojis/Objects/Gear.png" alt="Gear" width="25" height="25" /> Tech Stack</a>

- **[React](https://reactjs.org/)** – Powerful UI library for component-driven development.
- **[Tailwind CSS](https://tailwindcss.com/)** – Utility-first CSS for responsive, scalable styling.
- **[React Router](https://reactrouter.com/)** – Client-side routing for seamless navigation.
- **[Vite](https://vitejs.dev/)** – Lightning-fast build tool and dev server.
